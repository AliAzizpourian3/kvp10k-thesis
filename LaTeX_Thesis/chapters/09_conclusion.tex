\section{Summary of Contributions}
\label{sec:summary}

This thesis presented a comprehensive study on \kvp{} extraction from business documents using LayoutLMv3 with an attention-based linker. The main contributions are:

\begin{enumerate}
    \item \textbf{Dataset Analysis}: Characterized KVP10k dataset through layout clustering, revealing two distinct layout types and strong spatial proximity patterns in \kvps{}
    
    \item \textbf{Model Architecture}: Implemented LayoutLMv3 + Linker combining multimodal document understanding with explicit key-value association
    
    \item \textbf{Evaluation}: Established dual-threshold evaluation protocol (NED + IoU) and demonstrated \textcolor{red}{[DRAFT: Stage 4 performance results pending]} on benchmark
